1. Ambil screenshot sesuai daftar yang ada di `panduan-pengguna-lengkap.md`.
2. Simpan file PNG di folder ini dengan nama yang sama seperti yang direkomendasikan (mis. `01_login.png`).
3. Commit dan push ke repository setelah semua screenshot siap.
